## 🚀 Novidades e Otimizações

### 1. 📉 Redução Drástica de Tamanho
- O APK foi otimizado para a arquitetura **ARM64-v8a**, resultando em uma redução de tamanho de **51.9MB para apenas 17.8MB**.
- Isso torna o download e a atualização via OTA muito mais rápidos, economizando dados móveis dos usuários.

### 2. ✅ Performance
- Melhoria no tempo de resposta durante o processamento das fotos 3x4.
